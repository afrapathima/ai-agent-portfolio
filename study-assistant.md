# AI Study Assistant

## 1. Project Overview
The AI Study Assistant is a simple AI prototype designed to help students learn more effectively. It can summarize lecture notes, generate quiz questions, and explain difficult topics in simple English.

---

## 2. Key Features
- Summarizes long lecture notes into short points
- Generates multiple-choice quiz questions
- Explains complex topics in simple language
- Helps students revise quickly before exams

---

## 3. Sample Prompts and Responses

### Prompt 1:
User: Summarize this topic: Database normalization

AI Response:
Database normalization is the process of organizing data in a database to reduce redundancy and improve data integrity. It involves dividing large tables into smaller related tables.

---

### Prompt 2:
User: Generate 5 quiz questions about HTML

AI Response:
1. What does HTML stand for?
2. What is a tag in HTML?
3. What is the purpose of <head>?
4. Which tag is used for images?
5. What is a hyperlink?

---

### Prompt 3:
User: Explain JavaScript in simple words

AI Response:
JavaScript is a programming language used to make websites interactive, such as buttons, animations, and dynamic content.
