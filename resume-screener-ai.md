# AI Resume Screener

## 1. Project Overview
The AI Resume Screener is a simple AI prototype designed to evaluate job resumes and provide structured feedback. It helps simulate how AI can be used in HR systems to shortlist candidates based on skills, experience, and job requirements.

---

## 2. Key Features
- Analyzes resume content
- Matches skills with job requirements
- Provides improvement suggestions
- Gives basic candidate scoring (beginner level simulation)

---

## 3. Sample Prompts and Responses

### Prompt 1:
User: Evaluate this resume:
"I am an IT student with skills in HTML, CSS, JavaScript, PHP, and MySQL. I also know Photoshop and Illustrator."

AI Response:
Score: 7.5/10

Strengths:
- Good web development skills
- Knowledge of both frontend and backend
- Design skills are a plus

Improvements:
- Add real project experience
- Mention internships or practical work
- Improve CV structure and clarity

---

### Prompt 2:
User: Is this candidate suitable for a web developer internship?

Resume:
"Basic knowledge of HTML, CSS. No project experience."

AI Response:
Suitability: Moderate / Entry Level

Reason:
- Has basic technical skills
- Lacks project experience

Recommendation:
- Suitable for training-based internship only

---

### Prompt 3:
User: What skills should this candidate improve?

AI Response:
- JavaScript (advanced level)
- Real-world project building
- GitHub portfolio creation
- Backend development basics
