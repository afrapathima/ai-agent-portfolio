# Prompt Engineering Examples

## 1. What is Prompt Engineering?
Prompt engineering is the process of designing and improving inputs to get better AI responses.

---

## 2. Examples of Prompts

### Prompt 1
Basic: Explain AI  
Improved: Explain AI in simple words for a beginner student

Purpose: Make explanation easier  
Result: Clear and simple answer

---

### Prompt 2
Basic: List phones  
Improved: Suggest best smartphones under 100,000 LKR with reasons

Purpose: Get useful recommendations  
Result: Structured suggestions

---

### Prompt 3
Basic: Fix my code  
Improved: Find errors in this HTML code and explain step by step

Purpose: Better debugging help  
Result: Detailed explanation
